About

What is Project Antho?

Project Antho is an upcoming 2D Metroidvania title focused on hardcore, rhythmic parry combat. The game is set in an apocalyptic sci-fi world blended with elements of Khmer culture.

World Lore

Long ago, a massive disaster destroyed the world. Now, nature and advanced technology have grown together. Dark ruins are filled with glowing plants, high-tech machines, and Khmer-style structures. It is a beautiful but dangerous place to explore.

Hardcore Combat

  • Mechanics: Parries (Left, Right, Hold) + Counter Attacks.
  • Combat Style: Consistent, fast-paced rhythm fighting.
  • The Challenge: Surviving chaotic enemy patterns through pure reflex and precision.

Game Details

  • Genre: 2D Action Metroidvania / Rhythm Combat
  • Players: 1 Player (Single-player story campaign)
  • Platform: PC (Steam)
  • Input Support: Full Controller & Keyboard support

System Requirements

Minimum Specifications

  • OS: Windows 10 (64-bit)
  • Processor: Intel Core i3 / AMD FX-4350
  • Memory: 4 GB RAM
  • Graphics: Nvidia GTX 650 / AMD Radeon HD 7750 (1GB VRAM)
  • DirectX: Version 11
  • Storage: 2 GB available space

Recommended Specifications

  • OS: Windows 10/11 (64-bit)
  • Processor: Intel Core i5 / AMD Ryzen 3
  • Memory: 8 GB RAM
  • Graphics: Nvidia GTX 1050 Ti / AMD RX 560 (2GB VRAM)
  • DirectX: Version 11
  • Storage: 2 GB available space (SSD Recommended)
